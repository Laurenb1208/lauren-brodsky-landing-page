# Lauren Brodsky Personal Landing Page

## Description
This project is a personal landing page created for the BAIS 3300 Digital Product Management course at the University of Iowa. The website highlights my background, education, experience, and technical skills in a simple and professional format.

The goal of this project is to create a digital home where recruiters, hiring managers, and professional connections can quickly learn more about me and view my skills and experience.

## Features
- Hero section with profile photo and professional tagline
- About section describing my academic focus and career interests
- Education section highlighting my degree at the University of Iowa
- Experience section showing professional and project experience
- Skills section displaying technical tools and technologies
- Contact section with links to LinkedIn, GitHub, and email
- Responsive layout that works on desktop and mobile devices
