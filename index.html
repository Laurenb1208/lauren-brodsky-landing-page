SQL Oracle APEX Database Design Data Analysis Reporting

This project focused on building and analyzing a gym membership database using SQL and Oracle APEX. Reports and visualizations were created to show trends in member demographics, class participation, gym usage patterns, and trainer activity. The goal was to demonstrate how structured data and SQL queries can support data-driven decisions in a real-world business environment.
